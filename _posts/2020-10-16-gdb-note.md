# 补充用法

## 1 设置运行参数.

```bash
set args <your args>
```

*c*会运行到程序结束，*finish* 跳出 *step over(快捷键 s)* 进入的函数.  

## 2 vim 编辑代码

终端中输入 ```export EDITOR=vim```  后在 gdb 中输入 *edit* 即可在 vim 中打开文件.
